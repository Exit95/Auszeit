Seitenstruktur: Mein Vorschlag

Von 13 Sektionen auf einer Seite → fokussierte Startseite + eigene Unterseiten

Aktuell IST

Eine Seite, 13 Sektionen:

Hero Angebote Vertrauen Prozess Buchung Anfragen Workshops Galerie Gutscheine Bewertungen FAQ Kontakt Profil

Problem: Hohe kognitive Last. Nutzer scrollen ewig. Kein klarer Conversion-Pfad. Sektionen konkurrieren um Aufmerksamkeit.

Vorschlag NEU

Hybrid: Emotionale Startseite + fokussierte Unterseiten

Startseite als Appetitmacher, Details auf eigenen Seiten.

Startseite — emotionaler Einstieg, schnell zur Aktion

🎨

1. Hero

Großes Werkstatt-Foto, Headline, Subline, ein primärer CTA ("Termin buchen") + sekundärer ("Entdecke unsere Keramik"). Emotionaler Einstieg in 3 Sekunden.

✨

2. Was dich erwartet

Kurzvorstellung der Angebote: Keramik bemalen, Workshops, Events, Gutscheine. Jedes mit Bild + kurzer Text + Link zur Unterseite. Kein Preislisten-Gefühl.

📝

3. So funktioniert's

3 Schritte: Termin wählen → Keramik bemalen → Abholen. Simpel, vertrauensbildend.

📅

4. Termin buchen

Buchungskalender direkt auf der Startseite. Primäres Conversion-Ziel.

⭐

5. Stimmen unserer Gäste

2-3 Zitate, kompakt. Link zu mehr Bewertungen. Vertrauensaufbau vor Kontakt.

👩‍🎨

6. Über Irena

Kurzes Profil mit Foto. Persönlich, nahbar. "Ich freue mich auf dich."

📍

7. Kontakt & Anfahrt

Adresse, Öffnungszeiten, Karte, WhatsApp, Telefon. Footer-nah.

→ 7 Sektionen statt 13. Jede hat einen klaren Zweck. Scroll-Dramaturgie: Emotion → Angebot → Vertrauen → Aktion → Mensch → Kontakt.

Unterseiten — Tiefe für die, die mehr wollen

🖼️

/galerie

Vollwertige Galerie mit Story-Modus, Kategorien, Vollbild-Lightbox. Hier werden die neuen Fotos zum Star. SEO-Vorteil: eigene URL für Bilder-Suche.

🎓

/workshops

Alle Workshops mit Details, Preisen, Verfügbarkeit, Buchung. Eigene Seite = eigene SEO-Power ("Töpferkurs Gronau").

🎁

/gutscheine

Gutschein-Kauf mit Stripe, schön inszeniert. Eigene Seite = verlinkbar für Geschenk-Anlässe.

🎉

/events

JGA, Kindergeburtstage, Firmen-Events, besondere Anlässe. Anfrage-Formular. Zielgruppen-spezifische Ansprache.

❓

/faq

Häufige Fragen, ausführlich beantwortet. SEO für Long-Tail-Suchen ("Was kostet Keramik bemalen Gronau").

→ Bestehende Seiten bleiben: /admin, /gutschein/einloesen/[code], /stornierung, /gutschein-erfolg

Warum diese Struktur?

  • Conversion: Buchungskalender bleibt auf der Startseite — kürzester Weg zur Aktion
  • Kognitive Last: 7 statt 13 Sektionen = schneller verstanden, weniger Scroll-Müdigkeit
  • SEO: Eigene URLs für Galerie, Workshops, Events = mehr indexierbare Seiten, bessere Rankings
  • Zielgruppen: /events spricht Gruppen direkt an, ohne die Startseite zu überladen
  • Mobile: Kürzere Startseite = deutlich bessere mobile Experience
  • Navigation: Klare Menüstruktur statt endlosem Scroll mit Anker-Links
